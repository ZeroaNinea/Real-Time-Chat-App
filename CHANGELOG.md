# Changelog

## v1.0.0

- Authentication with pure JWT
- JWT key rotation
- Real-time messaging with Socket.IO
- GIF support via GIPHY
- Favorites
- Notifications
- User profiles: username, status, e-mail (optional, e-mail verification is not implemented), pronouns, bio, avatar
- Friends
- Private chat rooms
- Public chat rooms
- Channel creation
- Role-based permissions
- Markdown formatting
- Emoji support
- Avatar upload
- Chat room thumbnail upload
