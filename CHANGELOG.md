# Changelog

## v1.0.0

### Features

- Authentication with JWT
- JWT key rotation
- Real-time messaging with Socket.IO
- GIF support via GIPHY
- Favorites
- Notifications
- User profiles (username, status, email, pronouns, bio, avatar)
- Friends
- User blocking
- Private chat rooms
- Public chat rooms
- Channel creation
- Role-based permissions
- Markdown formatting
- Emoji support
- Avatar uploads via Cloudinary
- Chat room thumbnail uploads via Cloudinary
- Moderation and role assignment
